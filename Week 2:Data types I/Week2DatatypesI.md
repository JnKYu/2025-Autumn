
# CS112: Python 程序设计基础

## Week 2: 数据类型 I

### 1. Python 基础

* **语句 (Statement)**：由解释器直接执行的一条指令。

  ```python
  a = 23
  ```
* **表达式 (Expression)**：由操作数和运算符组成，会被求值并产生新结果。

  ```python
  6 * 7  # 结果: 42
  ```
* **变量 (Variable)**：存储数据值的容器。无需事先声明类型，赋值时自动创建。

  ```python
  x = 100
  x = "hundred"
  ```
* **赋值运算符**：`=`

  * 复合赋值：`+=`, `-=`, `*=`, `/=`, `%=`, `**=`, `//=`
* **标识符 (Identifier)**：变量/函数/类/模块的名字

  * 必须以字母或下划线开头
  * 不能使用空格或特殊符号
  * 不能是 Python 保留字（如 `if`, `while`, `class` 等）
* **最佳实践**：使用有意义的变量名，遵守 PEP 8 编码规范。

---

### 2. 基本数据类型

* **整数 (int)**：正数、负数或零

  * 运算符：`+ - * / // % **`
  * 优先级：`() > ** > * / % > + -`
* **浮点数 (float)**：带小数点的数值

  * 科学计数法：`3.2e5` → `3.2 × 10^5`
  * 常用函数：

    * `round(number, digits)` 四舍五入
    * `math.ceil()` 上取整
    * `math.floor()` 下取整
* **浮点误差**：由于计算机使用有限比特存储实数，可能出现近似误差。

---

### 3. 布尔值 (Boolean)

* **值**：`True`, `False`
* **比较运算符**：

  * `==` 等于
  * `!=` 不等于
  * `>`, `<`, `>=`, `<=`
* **逻辑运算符**：

  * `and` / `&` → 两个都为真才返回真
  * `or` / `|` → 至少一个为真则返回真
  * `not` → 取反

---

### 4. 字符串 (String)

* **定义**：由字符组成的有序序列，用 `'`、`"` 或 `''' """` 括起来。
* **基本操作**：

  * 拼接：`"Hello" + "World"`
  * 重复：`"Hi" * 3`
  * 比较：基于 ASCII 值比较大小
* **常用方法**：

  * `len()` → 字符串长度
  * `lower()`, `upper()` → 大小写转换
  * `replace(old, new)` → 替换
  * `count(value)` → 统计出现次数
  * `find(value)` → 查找子串位置
  * `split(separator)` → 拆分成列表
  * `startswith()`, `endswith()` → 判断前缀/后缀
  * `strip()` → 去掉首尾空白
* **索引与切片**：

  * 索引从 0 开始，负数从末尾计数
  * 切片：`s[start:stop:step]`

---

### 5. 类型转换

* `int ↔ float`
* `int/float ↔ str`
* `str → int/float`（必须是合法数字字符串）

---

### 6. 输出格式化 (Print Formatting)

* **格式化方式**：

  1. `%` 占位符：`%d %f %s`
  2. `format()` 方法
  3. f-String：

     ```python
     name = "Alice"
     age = 20
     print(f"学生 {name}, 年龄 {age}")
     ```
* **浮点数保留小数位**：

  ```python
  "{:.2f}".format(3.14159)  # 3.14
  f"{3.14159:.2f}"          # 3.14
  round(3.14159, 2)         # 3.14
  ```

---

### 7. 转义字符 (Escape Characters)

* `\n` 换行
* `\t` 制表符
* `\'` 单引号
* `\"` 双引号
* `\\` 反斜杠

---

### 8. 编程规范 (Good Practice)

* **代码可读性**：

  * 使用有意义的变量名
  * 遵循 PEP 8 风格（缩进、空格规范）
  * 添加注释：

    * 单行注释：`#`
    * 多行注释：`""" ... """`

---

Week 2 主要学习 **四种基础数据类型**（整数、浮点数、布尔值、字符串），理解变量与表达式的基本用法，掌握字符串操作与格式化输出，同时引入编程规范，为后续复杂编程打好基础。
